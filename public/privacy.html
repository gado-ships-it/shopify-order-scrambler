Privacy Policy

Order Scrambler — Last updated: May 2026

1. What we collect

Order Scrambler stores only the minimum data needed to operate:

  • Shop domain and Shopify access token — stored in a local SQLite database on our server. Used exclusively to call the Shopify Admin API on your behalf.
  • Order IDs and generated order names — stored for your own audit log (the “Recent scrambles” table). No order contents, customer names, or personal data are stored.

We do not collect, store, or share customer names, email addresses, payment data, or any other personal information.

2. How we use it

The access token is used solely to update the name field on new orders via the Shopify Admin REST API. No data is sold, shared, or used for marketing.

3. Data retention

Your shop’s access token is deleted within 30 days of app uninstallation (triggered by the shop/redact GDPR webhook Shopify sends us). Scramble log entries are retained for 90 days then automatically purged.

4. GDPR / data subject requests

Order Scrambler does not store personal data about end-customers. If you receive a data subject access or erasure request related to this app, contact us and we will assist. Since we hold no customer PII, our response will confirm that no data is held.

5. Security

Access tokens are stored in a server-side SQLite database, not exposed to browsers. The server uses HTTPS (TLS 1.2+). Webhook payloads are validated with HMAC-SHA256 before processing.

6. Third parties

We use Shopify’s Admin API to update order names. No other third-party services receive your data.
